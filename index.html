🧩 5단계: 원소와 화합물

🤔 생각해 봅시다

물은 수소와 산소로 분해할 수 있었습니다. 그렇다면...

"수소"나 "산소"는 더 분해할 수 있을까요?

과학자들이 아무리 노력해도 수소와 산소는 더 이상 다른 종류의 물질로 분해할 수 없었습니다.

정리:
• 원소 — 한 종류의 원자로 이루어진 물질, 더 이상 다른 종류의 물질로 분해되지 않음
• 화합물 — 두 종류 이상의 입자가 모여 만들어진 물질

🧱 6단계: 원소 vs 원자

🤔 원소와 원자, 어떻게 다를까?

"원소"와 "원자"는 비슷하게 들리지만 서로 다른 개념이에요. 레고에 비유해 봅시다!

원소 = 블록의 종류
"빨강, 파랑, 노랑 — 3종류"
→ 성분이 무엇인지 물을 때
원자 = 블록 알갱이(개수)
"빨강 2개 + 파랑 3개 = 5개"
→ 알갱이가 몇 개인지 물을 때

💧 물(H₂O)로 예를 들면?

성분의 종류를 물을 때
2가지 원소
수소(H), 산소(O)
알갱이 개수를 물을 때
3개 원자
H 2개 + O 1개

📋 실험 정리 — 나만의 노트 만들기

✏️ 빈칸에 직접 타이핑하여 오늘 실험을 정리해 보세요!

1. 물의 전기 분해

물에 를 흘리면 (−)극에서 기체가, (+)극에서 기체가 발생한다.

2. 물의 성분

물(H₂O)은 (H)와 (O)로 이루어져 있다.

3. 원소와 화합물

원소: 한 종류의 로 이루어진 물질로, 더 이상 다른 종류의 물질로 되지 않음.
화합물: 종류 이상의 입자가 모여 만들어진 물질.

4. 원소와 원자

원소: 물질을 이루는 기본 성분의 를 말하는 개념.
원자: 물질을 이루는 알갱이 하나하나()를 말하는 개념.

2H2O → 전기 분해 2H2 + O2
